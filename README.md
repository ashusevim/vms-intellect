# Visitor management System

Tech stack

- front-end: Angular
- back-end: springboot

# How to setup the project?

## For frontend

1. Clone the repository
```
git clone https://github.com/ashusevim/vms-intellect
```

2. go to frontend folder

```
cd frontend
```

3. install all the necessary dependencies

```
npm install
```

4. Run the app

```
npm start
```
